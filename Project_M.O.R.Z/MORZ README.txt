# Introduction
- Top-down tank style video game created by a peer and I.

## About
Project M.O.R.Z is a game that takes place on a fictional planet where you must use your tank to destroy aliens and buy upgrades!

### Installation
- Built for use on windows devices.

- Download the executable, run the install wizard and enjoy the gameplay!
